Forms

We can use connected lists to implement palettes and canvases where you can drag items from a palette onto a canvas to add the items to the canvas. Consider the example below with a palette of form elements on the left and a canvas on the right. This is an examped version of an earlier example illustrating connecting draggable and sortable lists. Here they implement a simple form building app where you can add form elements from the palette onto a form. The form items can then be repositioned.

The script for connecting the draggable elements to the sortable elements stayed exactly the same. No added complexity. What did change was the actual elements being dragged, dropped, and sorted. These are now sets of related form elements that together can build any type of form. 

	$(function() {
		$("#palette > li").draggable({
			connectToSortable : "#form",
			helper:"clone"
		});
		$("#form").sortable({
			"axis" : "y"
		});
	});

The list of draggable items now defines a palette of web form elements such as input fields, textareas, radio buttons, checkboxes, and buttons. Styling is used to hide label input fields on the palette, and to display components more compactly.
