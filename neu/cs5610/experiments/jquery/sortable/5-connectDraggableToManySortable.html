Conneting Draggable to Several Sortable

Let's go back to our simpler example where we connected a generic palette on the left to a sortable list on the right. We want to be able to handle not just one form, but several forms on the same page. We should then be able to drag from the same palette onto various forms. The example below illustrates this functionality. Drag from the blue palette on the left onto any of the three sortable lists on the right. The only manjor difference between one sortable list or many sortable lists is that I am using classes instead of IDs to refer to the lists. That is, we don't have a single list now with ID #sortable. We now have several lists that belong to the same class .sortable. Below is the new script that connects the palette to the lists. Notice how #draggable and #sortable have been changed to .draggable and .sortable. 

	$(function() {
		$(".draggable > li").draggable({
			connectToSortable : ".sortable",
			helper:"clone"
		});
		$(".sortable").sortable({
			"axis" : "y"
		});
	});
