Conneting Draggable to Sortable

Draggable items can be connected to Sortable lists so that we can use some of the behavior of draggables. The left list below contains draggable items. The list on the right is sortable. Try dragging the items on the left onto the list on the right. Notice how the original draggable items do not move. A "helper" draggable item is created as it is dragged onto the right list.

Both draggable and sortable are implemented as list as shown below: 

	<ul id="draggable" class="nobullets fleft sortable">
	  <li class="bluebg">Item A</li>
	  <li class="bluebg">Item B</li>
	  <li class="bluebg">Item C</li>
	  <li class="bluebg">Item D</li>
	  <li class="bluebg">Item E</li>
	</ul>
	 
	<ul id="sortable" class="nobullets fleft sortable">
	  <li class="redbg">Item 1</li>
	  <li class="redbg">Item 2</li>
	  <li class="redbg">Item 3</li>
	  <li class="redbg">Item 4</li>
	  <li class="redbg">Item 5</li>
	</ul>

The script below configures the list items of the left list as draggable and the list on the right is configured as sortable. The draggables on the left are connected to the sortable list, and they use a cloned copy of the draggable items instead of the original list item. 

	$(function() {
		$("#draggable > li").draggable({
			connectToSortable : "#sortable",
			helper:"clone"
		});
		$("#sortable").sortable({
			"axis" : "y"
		});
	});
