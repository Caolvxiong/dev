Hover

Use hover event handlers to handle hover gestures. To define hover handlers, use the hover() method on the element of interest. Provide methods to be invoked when the element is hovered on or hovered off. Hover on and off this text paragraph to change it's background color.

The script below binds a hover event on elements with the hover class, e.g., this and the paragraph above. The hover method takes 2 callbacks. The first is called when the element is hovered on, and the second is called when the element is hovered off. 

	$(function() {
		$(".hover").hover(makeBlue, makeRed);
	});

In this case callback methods makeBlue() and makeRed() do what their names imply, they turn the element to blue or red. Below is their implementation. 

	function makeBlue() {
		var element = $(this);
		element
			.css({
				"background-color" : "blue",
				"color" : "white"
			});
	}
	function makeRed() {
		var element = $(this);
		element
			.css({
				"background-color" : "red",
				"color" : "white"
			});
	}
