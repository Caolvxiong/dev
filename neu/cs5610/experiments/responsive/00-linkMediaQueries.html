Link Media Queries

This document changes background color as you resize the browser window. To achieve this we load several CSS style sheets, but make them active only at different window sizes. The three style sheets are:

  • 00-style0.css
  • 00-style1.css
  • 00-style2.css and
  • 00-style3.css
Their content is:

00-style0.css

.main {
    font-size:30px;
    height : 100px;
}

00-style1.css

.main {
    background-color : red;
}

00-style2.css

.main {
    background-color : green;
    height : 400px;
}

00-style3.css

.main {
    background-color : yellow;
}

Media Queries

We load all 4 sheets at once but they don't all apply at the same time. We conditionally apply the sheets by using media queries. The sheets are loaded as follows: 

<link rel="stylesheet" type="text/css" href="00-style0.css" />
<link rel="stylesheet" type="text/css" media="(min-width: 50px)  and (max-width: 400px)" href="00-style1.css" />
<link rel="stylesheet" type="text/css" media="(min-width: 401px) and (max-width: 800px)" href="00-style2.css" />
<link rel="stylesheet" type="text/css" media="(min-width: 801px)" href="00-style3.css" />

The first style sheet 00-style0.css has no conditions and is always active.

The other 3 style sheets all have media rules that govern when they are active. The rules are predicates that evaluate browser attributes using boolean expressions and activate the stylesheet when the expressions are true.

The second style sheet 00-style1.css is active only when the window's width is between 50 and 400 pixels.

The third style sheet 00-style2.css is active when the window is between 401 and 800 pixels.

The last style sheet 00-style3.css is active when the window is at least 801 pixels wide.
