Draggable

This example shows you how to use jQuery's draggable functionality. Consider the two DIVs below


These have been declared as follows:

        <div class="blue w100 h100 draggable"></div>
        <div class="red w100 h100 draggable"></div>

Using jQuery, we can grab the DIVs by their class as follows:

        $(function () {
            $(".draggable").draggable();
        });

We add the draggable functionality to the element by envoking .draggable() on it. Note that the class "draggable" is not what makes the DIVs draggable. These classes are merely used to be referenced by jQuery in $(".draggable"). Once we have the DIVs in the jQuery context, we can invoke any of a multitude of functions declared by jQuery. The draggable() function adds the ability to drag the DIVs.
