Adding Draggables on Demand

This example shows how to use jQuery's to add draggable blocks on demand. Consider the link below:

Add Another Block
 
 

Click on the Add Another Block link above and notice how it adds more and more blocks from top to bottom and left to right. Try dragging the new blocks. Cool huh? The link's html is as you would expect. We are giving it the id 'add' so we can grab it with jQuery: 

	    	<a href="#" id="add">Add Another Block</a>

We grab the link from its ID and assign it a click handler as follows:

	        $(function () {
	            $("#add").click(addAnotherBlock);            
	        });

The event handler dynamically creates a DIV block content, makes draggable, and adds it to a DIV called canvas 

    		function addAnotherBlock() {
        		$("<div class='h100 w100 redbg fleft'>")
	        		.draggable()
	        		.appendTo("#canvas");
        	}
