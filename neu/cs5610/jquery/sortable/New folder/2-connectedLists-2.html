Connected Sortable Lists

Sortable lists can be connected to one another so that items can be sorted between them. The two sortable lists below are connected. Try dragging elements from one onto the other. You can drag both ways.

  • Item A
  • Item B
  • Item C
  • Item D
  • Item E
  • Item 1
  • Item 2
  • Item 3
  • Item 4
  • Item 5

The lists are implemented as simple un-ordered lists styled with no bullets and floated left so that they show side by side 

	<ul id="sortable1" class="nobullets fleft sortable">
	  <li class="bluebg">Item A</li>
	  <li class="bluebg">Item B</li>
	  <li class="bluebg">Item C</li>
	  <li class="bluebg">Item D</li>
	  <li class="bluebg">Item E</li>
	</ul>
	 
	<ul id="sortable2" class="nobullets fleft sortable">
	  <li class="redbg">Item 1</li>
	  <li class="redbg">Item 2</li>
	  <li class="redbg">Item 3</li>
	  <li class="redbg">Item 4</li>
	  <li class="redbg">Item 5</li>
	</ul>

To connect the lists, just use the connectedWith option of the sortable jQuery plugin as shown below. Here, both lists are connected to the other. 

	$(function() {
		$( "#sortable1" ).sortable({
			connectWith: ".sortable"
		});
		$( "#sortable2" ).sortable({
			connectWith: ".sortable"
		});
	});
