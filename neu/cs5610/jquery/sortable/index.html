Sortable

This example shows you how to use jQuery's sortable functionality. Consider the following list:

  • Lorem ipsum dolor sit amet, consectetur adipisicing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua
  • Ut enim ad minim veniam, quis nostrud exercitation ullamco laboris nisi ut aliquip ex ea commodo consequat
  • Duis aute irure dolor in reprehenderit in voluptate velit esse cillum dolore eu fugiat nulla pariatur

Now, try dragging the list items and repositioning them between other members of the list. Note how the other members move away to make space for the list item you are dragging. When you drop the list item, it inserts itself in the new position. Isn't that cool!?

The HTML has been declared as follows:

	<ul class="sortable">
		<li class="red">Lorem ipsum dolor sit amet, consectetur adipisicing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua</li>
		<li>Ut enim ad minim veniam, quis nostrud exercitation ullamco laboris nisi ut aliquip ex ea commodo consequat</li>
		<li class="blue">Duis aute irure dolor in reprehenderit in voluptate velit esse cillum dolore eu fugiat nulla pariatur</li>
	</ul>

Using jQuery, we can grab the list by its class as follows:

        $(function () {
            $(".sortable").sortable();
        });

We add the sortable functionality to the list by envoking .sortable() on it. Note that the class "sortable" is not what makes the DIVs sortable. These classes are merely used to be referenced by jQuery in $(".sortable"). Once we have the DIVs in the jQuery context, we can invoke any of a multitude of functions declared by jQuery. The sortable() function adds the ability to drag the list items in the list.
